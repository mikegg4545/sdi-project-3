# Asset Manager (Project 3)

A full-stack application for managing financial assets and their categories.

## Features

- Create, read, update, and delete assets
- Assign assets to categories
- View all assets in a simple UI
- Backend API with RESTful routes
- PostgreSQL database integration

## Tech Stack

**Frontend**
- React (Vite)

**Backend**
- Node.js
- Express

**Database**
- PostgreSQL
- Knex
